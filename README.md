# yandex-mail-pdfSender-maxbot
# 📩 Yandex Mail → PDF Sender → MAX Bot

Automated system that reads emails from Yandex Mail (IMAP), extracts PDF attachments, parses invoices, and sends structured data + files to MAX messenger bot (Telegram Bot) (Go integration).

---

# 🚀 Project Purpose

This project automates:

- 📬 Receiving emails from Yandex Mail
- 📎 Extracting PDF attachments
- 📄 Parsing invoices from PDF
- 🤖 Sending data to MAX messenger via Go bot
- 📊 Structuring invoice data for further processing
